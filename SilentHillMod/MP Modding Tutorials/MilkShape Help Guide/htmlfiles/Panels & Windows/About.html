The About Window

The about window contains the credits for the creation of MilkShape 3D program, the registration feature and information about the computer it is running on.

System Information

The system information the about box provides is

  • Free Memory (RAM) in megabytes
  • Total Memory (RAM) in megabytes
  • Memory Usage (RAM) as a percentage
  • Free Disk Space (HDD) in kilobytes
  • Total Disk Space (HDD) in kilobytes
  • OpenGL Vendor
  • Renderer (Graphics Card)

Registration

When MilkShape 3D is launched on the first day of MilkShape 3D being installed on your computer you will be asked to make sure the date and time is correct then you will be presented with the following dialog box.

If the copy of MilkShape 3D you are running has not been registered then the register button at the bottom right of the About box will be enabled. Clicking this button shows the window in the following image.

Into this box you must enter your name (the name you provided Mete Ciragan with for registration) and the registration code that was sent to you by Mete Ciragan. See Registration for more details. Provided the information is correct, press ok and the dialog in the following image will be displayed.

This will show you the registration information you have entered, to accept the registration for the shown details press the 'Yes' button and the dialog in the following image will be displayed.

Upon this message appearing press the 'OK' button, then MilkShape 3D will tell you that it is shutting down to complete the registration process. Just start MilkShape 3D back up and you will have a registered copy.
