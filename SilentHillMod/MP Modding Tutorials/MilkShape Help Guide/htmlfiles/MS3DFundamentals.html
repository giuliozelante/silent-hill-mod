MilkShape 3D Fundamentals

The Interface of MilkShape 3D has been designed for ease of use and maximum productivity.

There are three main parts of the MilkShape 3D interface, these are:

  • Menus
 Menus provide a worded identification of tools that are not regularly accessed.
  • Toolbox
 The Toolbox allows fast access to those tools used frequently and is primarily involved in the construction of models.
  • Keyframer
 Used solely for the purpose of animating. Keyframes are set to store the positions of a skeleton over time.

All the features in MilkShape 3D are accessed through a combination of these three main interface components.
