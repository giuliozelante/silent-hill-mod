Viewport Popup Menu

The viewport popup menu can be accessed by right-clicking in any of the viewports (3D or 2D). The menu provides a set of options that allow the current view to be altered. Below is an image showing the menu, click on one of the options to see a description for that menu item.

Shading Type

Wireframe, Flat Shaded, Smooth Shaded and Textured are all shading types.

Wireframe

This shows the model as it would be composed with lines between the vertices to represent the outlines of the faces of the model.

Flat Shaded

This shows the model with single colour flat faces to the structure. In this view, individual polygons can be seen easier than in the other views.

Smooth Shaded

Smooth shading creates a gradient polygon when the light determines the brightness of the polygon at each vertex it belongs to. This gives a natural look, not possible in wireframe or flat shading.

Textured

This shading type is the same as smooth shading but assigned textures and alphamaps are also displayed.

Projection

The projection sub menu brings up the menu shown in the image below.

This menu choice allows you to change the perspective of the camera. All views listed apart from the 3D view use Orthographic projection (3D uses perspective). This allows you to control which way you view the object from allowing for easier modelling. This can also be changed through the Viewport Caption.

Show Axis

This option allows you to turn the display of the 3 axis lines in the centre of the view on and off. By default these lines are X = Yellow, Y = Turquoise and Z = Pink. In the orthographic (2D) views there will only be 2 lines displayed as the third lies on the same plane as that of the camera.

Show Grid

This option toggles on and off the display of the grid lines in the viewport. This option can be used in all viewports, 3D and 2D (individually) to toggles grid lines on and off.

Draw Backfaces

All faces that face inward (have a clockwise vertex order) will not be displayed when this option is off. Inward facing faces are ones that are completely black when flat shadin is turned on.

Reset View

This menu option allows the user to reset the viewport back to the default viewport settings. The defaults are the settings used when MilkShape 3D is started up.

Frame All

This will frame the complete model (including all polygons/groups in the current workspace) into the centre of the view. This allows the easy viewing of all parts of the current model, it can also be useful when you are finished working on a specific part and instead of resetting the view to default you can frame the entire model for a better view.

Frame Selection

This works in much the same way as Frame All except the view is focused on the current selected object/s in the workspace.

Choose Background Image...

This allows the user to choose a background image for their model. This feature is commonly used to have a reference picture from which to work. This option will open a standard windows open box, choose your image from the appropriate directory on disk. A background image cannot be applied to a 3D (perspective) viewport.

Each pixel in the background image takes up 0.25 units on the MilkShape 3D orthographic grid, so for example a 100 x 100 pixel image will take up 25 x 25 units on the grid. The background image is always centered, if the image is of even height or width then the best estimate is used.

Take Screenshot...

This menu choice will bring up a standard windows save box.The current contents of the viewport can be saved in .bmp format. When using this function you must type the .bmp file extension onto the end of the filename, if this is not done it will be an unknown type file.

Record AVI...

This feature can be used to record an AVI format file of the animation of the currently loaded model. The AVI will run for the entire length of the frames currently specified in the Keyframer at the bottom of the MilkShape 3D window. If you have an animation then it will be played and captured in the AVI file, if there is not animation then you will just get a still model, i.e. no animation or movement.

Maximize

This feature simply toggles the viewport between maximised and minimised. When maximised the viewport takes up the space that all viewports would have taken up.
