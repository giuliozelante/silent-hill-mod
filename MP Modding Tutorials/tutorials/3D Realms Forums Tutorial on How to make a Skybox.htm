Since you have 3dsMax 4, I went ahead and uploaded my template.

You can download it here:

http://hellskitchen.paynereactor.com/Hell's%20Kitchen_files/main_files/Bonus/Worldsphere.max

When you open the file, it will give you an error about a blue sky texture not found -- that's because I didn't include the actual texture.

Just go into the materials dialogue (the icon is 4 spheres, at the top right of the screen -- use the help docs if you can't find it), and change the texture to your own one (it's the first sphere in the list - top left). Just export as a kf2 to the worldspheres directory with the export options left as they are (geometry, animation, materials checked).

Finally, add a new entry in the worldspheres.txt, and assign it to your level in the levels.txt. It should work, but post here if you get stuck.

Autch, I feel sort of embarassed here. No-one emailed me on the subject, trust me! And I don't get to read everything on the forums. Argh! Anyway...

If you export the worldsphere animation with animation checked from the KF2 export dialog (obviously should be looping) and make your worldspheres entry look as something like:
code: 
[generic_a]             // Light weather, middlegrey
{
        [Geometry]
        ExportData = "files\skybox_generic_a.kf2";

        [Animation]
        Index = 0;
        Filename = "files\skybox_generic_a.kf2";
}
It should work like a charm.
