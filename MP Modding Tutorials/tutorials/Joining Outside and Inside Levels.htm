Introduction
Author: *rrr*bbb*
Date: 10/28/01
Source of the german original here
Translation by zephyrin_xirdal 09/05/02

Preface
After the tutorial how-to-create-an-outsidelevel now this one, which explains how to join an insidelevel with the outsidelevel. I am using the map from the outsidelevel-tutorial.

Door Frame
At first we need a frame for the door. One can do it without this, but it looks better with a frame.

Align the grid to one of the outsidewalls of the level (Extrude-mode, hover over poly, hit "A"). A door in Max Payne has the standard-dimensions 1.5x2.5 meters. Create a mesh with the dimensions 2.5x3.0. This way we'll have 0.5m space to each side.



Now move the mesh to the back - just as far as its thickness, so it will just touch the outsidemap. Now union it to the room via boolean operation. Align the grid anew.



Now for the entrance. Create a mesh at the middle of the recess/doorframe which has the dimension 1.5x2.5m. Move it back and union it to the room.


Room Creation
As we've got an entrance now, we are going to create a room adjoining to it. This room is now unioned to the outsidemap. Tada - the connection between inside and outside.


Window Creation
At first we create two windows which are one story above the entrance. Align the grid again, create two windows (two times as thick as the meshes at the entrance were), move them to the back and connect them with the outside-world via union.



Create a room again and union it.

Final
At the end you join the rooms with a room between, and maybe you create a staircase. Set the exits and you're done.
