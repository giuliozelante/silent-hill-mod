Creating Local based hits By: sentinel







How to create Locational based damage effects
This tutorials shows you how to make local based damage effects, as seen in Suicidal Solutions ;).
First of all, you need to know how the bones are called where you want the hit effect.
For example, if you want effects on the head, i recommend using these bones:
-Bip01 Head
-Bip01 Neck
-Bip01 Spine1

If you don't want to make them on the head, and don't know how the bones are called on the place you DO want,
open up database\skeletons\male.txt.
Scroll down to the bottom, and you can see a list of bone names.
Let's start.

Begin with opening up the text file of the desired skin that you want the hit effect on.
For example, C03_Semicleaner_A.txt.
Scroll down a bit till you see [Target] blocks, or search with CTRL+F for [Target].
As you can see, these blocks have bones on them, these bones define what type target is, what material it has, the damage multiplier (which multiplies the damage done, by the number defined, if the character is hit on that place. Most of the times, there stands no number, but something like MULTIPLIER_CHARACTER. This means that it is defined in another file, which is included in this file, to check what that file is, go way up to the absolute top, and you'll see lines like "#include targetid.h" Targetid.h is the file that defines it in this case)
and the bones the [Target] block affects.
There are three types of target's which you can use:
-Character
-Helmet
-Bodyarmor

To do the specific hit effect, place a [Target] block, which uses the Material Bodyarmor, the Target TARGET_BODYARMOR, and multiplier MULTIPLIER_DEFAULT.
In this example, we are using C03_Semicleaner_A.txt and make the hit effects on the head.
These are the semicleaner_a's target blocks:

[Target]
{
[General]
Name = "Torso";
Type = TARGET_BODYARMOR;
NoPenetrationMultiplier = MULTIPLIER_BODYARMOR;
FullPenetrationMultiplier = 1.0;
Material = "character_bodyarmor";

[Bone] Name = "Bip01 L Clavicle";
[Bone] Name = "Bip01 R Clavicle";
[Bone] Name = "Bip01 Spine1";
[Bone] Name = "Bip01 Spine";
}

[Target]
{
[General]
Name = "Body";
Type = TARGET_CHARACTER;
NoPenetrationMultiplier = MULTIPLIER_DEFAULT;
FullPenetrationMultiplier = 1.0;
Material = "character";


[Bone] Name = "Bip01 Neck";
[Bone] Name = "Bip01 Pelvis";
}

[Target]
{
[General]
Name = "Head";
Type = TARGET_CHARACTER;
NoPenetrationMultiplier = MULTIPLIER_HEAD_NO_ARMOR;
FullPenetrationMultiplier = MULTIPLIER_HEAD_NO_ARMOR;
Material = "character";

[Bone] Name = "Bip01 Head";

}

[Target]
{
[General]
Name = "ArmsAndLegs";
Type = TARGET_CHARACTER;
NoPenetrationMultiplier = MULTIPLIER_ARMSLEGS_NO_ARMOR;
FullPenetrationMultiplier = MULTIPLIER_ARMSLEGS_NO_ARMOR;
Material = "character";

[Bone] Name = "Bip01 L UpperArm";
[Bone] Name = "Bip01 L Forearm";
[Bone] Name = "Bip01 L Hand";
[Bone] Name = "Bip01 L Thigh";
[Bone] Name = "Bip01 L Calf";
[Bone] Name = "Bip01 L Foot";
[Bone] Name = "Bip01 L Toe0";
[Bone] Name = "Bip01 R UpperArm";
[Bone] Name = "Bip01 R Forearm";
[Bone] Name = "Bip01 R Hand";
[Bone] Name = "Bip01 R Thigh";
[Bone] Name = "Bip01 R Calf";
[Bone] Name = "Bip01 R Foot";
[Bone] Name = "Bip01 R Toe0";
}


In order to place the hit effects on the head, we need to change it to this:

[Target]
{
[General]
Name = "HitEffect";
Type = TARGET_BODYARMOR;
NoPenetrationMultiplier = MULTIPLIER_DEFAULT;
FullPenetrationMultiplier = 1.0;
Material = "character_bodyarmor";

[Bone] Name = "Bip01 Spine1";
[Bone] Name = "Bip01 Neck";
[Bone] Name = "Bip01 Head";
}

[Target]
{
[General]
Name = "Body";
Type = TARGET_CHARACTER;
NoPenetrationMultiplier = MULTIPLIER_DEFAULT;
FullPenetrationMultiplier = 1.0;
Material = "character";


[Bone] Name = "Bip01 Pelvis";
}

[Target]
{
[General]
Name = "ArmsAndLegs";
Type = TARGET_CHARACTER;
NoPenetrationMultiplier = MULTIPLIER_ARMSLEGS_NO_ARMOR;
FullPenetrationMultiplier = MULTIPLIER_ARMSLEGS_NO_ARMOR;
Material = "character";

[Bone] Name = "Bip01 L UpperArm";
[Bone] Name = "Bip01 L Forearm";
[Bone] Name = "Bip01 L Hand";
[Bone] Name = "Bip01 L Thigh";
[Bone] Name = "Bip01 L Calf";
[Bone] Name = "Bip01 L Foot";
[Bone] Name = "Bip01 L Toe0";
[Bone] Name = "Bip01 R UpperArm";
[Bone] Name = "Bip01 R Forearm";
[Bone] Name = "Bip01 R Hand";
[Bone] Name = "Bip01 R Thigh";
[Bone] Name = "Bip01 R Calf";
[Bone] Name = "Bip01 R Foot";
[Bone] Name = "Bip01 R Toe0";
}

Make the changes and save the file.
Once you did this, you are done editing the skin's file. Next you need to edit the projectiles.
Open up the desired projectile you want to use the hit effect with, for example BulletDesertEagleSingle_Player.txt.
The whole file can stay intact, only two defines matter here.
Which are:
DamageType and DamageTypeWithArmor.
You probably understand where i'm going.
You need to edit the DamageTypeWithArmor's value, which is now, DAMAGETYPE_SMALL.
Change it to an un-used damagetype, for example DamageType_door.
Which would make DamageTypeWithArmor = DAMAGETYPE_DOOR;
Save the file and move on.

Now, you need to edit the semicleaner's anims, open up database\skeletons\male.txt.

Search with CTRL+F for CHARANIM_DOORDAMAGE. You can give the message for the desired effect there.
And also edit the FileName.
Save the file once you've made the changes, and TEST!
