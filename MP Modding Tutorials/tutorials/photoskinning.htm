Photoskinning with MP2 By: GamesFreak


Photo Skinning Tutorial

This tutorial will explain how to skin characters with the photo skinning technique
for Max Payne 2. You will learn how to make the best out of the worse photos.


What do you need?


- Max Payne 2 Developer tools installed on your PC.
- Photoshop (Pictures are taken from Version 7.0).
- DDS format plug-in for Photoshop installed and runs fully. (Download
here).
- Basic knowledge in Photoshop.


Setup:


- Turn on your copy of Photoshop, click: File->Open, then choose the file
C:\MaxPayne2Dev\Game\data\database\skins\Sharedtextures\MaxPayne_Face. Max's
face skin will open.



- Usually, Pictures from the side of the face are hard to find so I will show
how to do it with a front picture. I will use a bad quality Arnold
Schwarzenegger picture I found to skin with. Open the picture like in the
previous step.


Let's Start:


- Cut Arnold's face to two, right in the middle, don't be afraid to cut some thing you don't need.
Copy the left side and paste it on Max's face, place it like that, so Max's chin will be in the same place as Arnold's chin.
Now go to Edit->Free transform. Transform Arnolds half face so it will look
like in the next picture.



- Start erasing unneeded parts like the ear and the hair (of Arnold). In the right bottom of the screen there is the "Layers" window, drag the above layer to the small icon which creates a new layer (looks like a paper),
this will duplicate the layer, now transform (like in previous stage) the duplicated layer and flip it to the other side, like a mirror effect.



- Now press on the duplicated layer and press Ctrl+E, this has merged both of Arnolds face parts. Lower the opacity and transform the face so the eyes, nose and mouth will match to Max's face.
Start erasing the line in the center of the face, do that by using the clone tool, or eye dropping the skin color and using the brush to color the line.
You can use the smudge tool to change the brows or mouth to change face expressions.
You will end up with something like this:



- Turn the Smudge tool and start smudging of Arnolds face, the strength should be up to 50 %. Eventually you will get something like this:



- We are almost finished. The last thing we need to do is to set that Arnold's and Max's colors will be the same. We do that with HUE/Saturation tool, Brightness/Contrast tool, and opacity changing.
Check Arnold's layer, now go to Image->Adjustment->Hue/Saturation.
Set the properties like here:



- In this case, these are the properties but in other cases you will have a different picture with different colors, then,
do what I did to set the color right:
See what color Max is, if he is more red as example, then move the hue bar to make the picture more red, then move the saturation bar to increase or decrease the color strength, and finally the lightness bar, to change the lightness.
Back to our case, you probably got something like in this picture:


- To make the skin better, "play" with his body parts like, color his hair to more orange, make a squared chin etc. the best result I got (in 5 minutes) is:




Saving:


- Go to File->Save As, save your skin as PSD wherever you want. Then do that again, but in the format option, choose DDS (*.DDS).
A window will pop up, set it's preferences like in this picture:



Then click save.


- Now rename the file to "MaxPayne_Face.dds" and place it in data/database/skins/sharedtextures,
overwrite the existing file (don't forget to backup!!), that's it! Now run your Game/Mod! You're done!
