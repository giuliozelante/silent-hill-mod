Re: How do to get world spheres to work
    07/03/03 03:26 AM

It's pretty simple:

Create a material category called Skybox, and add the blue/cyan texture called skybox (I think an ordinary texture works, but you're best off using the special skybox one).

Now texture all the areas that you want to appear as sky with this new texture.

Now, in \database\levels\levels.txt look for the entry for your new level, and find the value labelled "worldsphere = "

Now just enter the name of a worldsphere,

eg:

worldsphere = "sunset";

Other values you can use are:

intro,
asgard
coldsteel
sunset
docks
lowereastside
skyscrapers_streetlevel
skyscrapers_midheight
skyscrapers_penthouse
skyscrapers_penthouse_high
bronx
