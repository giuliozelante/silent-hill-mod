The Message Panel

The message panel is a useful feature to have when you are creating models in general and when you are trying to export for games such as Half-Life, Quake II, Quake III, etc...

The message window outputs messages relating to the operation the user is currently trying to perform. Most operations in MilkShape 3D are displayed in the message panel, it can be a useful tool while modelling for beginners as it outputs error messages generated while performing certain operations. Experienced users may only use it when exporting to game formats because it can take up workspace.
