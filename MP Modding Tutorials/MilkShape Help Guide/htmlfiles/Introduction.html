Welcome to MilkShape 3D

MilkShape 3D allows the easy creation and manipulation of polygonal characters and objects through the use of tools which are both intuitive and easy to access.

This Help documentation aims to help users of MilkShape 3D, both experienced and inexperienced alike. Every feature of the program is explained in full as well as their uses and a demonstration in words and pictures as how to achieve the best results while using them. There are some topics (such as animation) that are written and presented in the style of tutorials, these aim to have you pick up the functionality easily. Hyperlinks are appropriately placed throughout the Help documentation so that you can check relevant topics without the need to search for them. To aid understanding, pictures have been included, where relevant to show how to access certain features as they are explained.

May you have fun and great success in your use of MilkShape 3D.
